Base use

The usage is almost the same as otherCSS In JSframeworks

Supports special nested selectors`&`andCSSall native selectors, and also provides a polymorphic`as`property, used to define the label that the component renders

Also, if usingTypescript, prop definitions are updated when`as`props are added 

import { styled } from '@styils/react';

// We can add namespce which may be useful when developing class libraries
const Image = styled({tag:'img',namespce:'label'},{ ... })

const Button = styled('button', {
  // :Styles inside global will generate global styles
  ':global':{
    body:{
      ...
    },
  }
  fontSize: '13px',
  padding: '10px 15px',
  '&:hover': {
    fontSize: 18, // The default unit is px
  },
  '& .foo':{
    ...
  },
  [`& ${Image}`]:{
    ...
  }
});

render(<Button as="a" href="google.com"><Image/></Button>)

Dynamic rendering

Style components are created with avariantsattribute

You can define a single dynamic rule, multiple dynamic rules, or even compound dynamic rules 

import { styled } from '@styils/react'

const Button = styled(
  'button',
  {
    fontSize: 14
  },
  // variants is the third parameter of styled
  {
    size: {
      small: {
        fontSize: 12
      },
      large: {
        fontSize: 16
      }
    }
  }
)

render(<Button variants={{ size: 'small' }}>Button</Button>)

Dynamic interpolation replaces props props passing, because passing values through props will bring a large performance overhead, in addition, we may write the following code. 

const Side = styled('div', () => ({
  transform: translateX(`${props.move}px`);
}))

function foo() {
  // Each setMove will create multiple almost identical CSS rules
  // in this case we should use style
  const [move, setMove] = useState(0)
  return <Side move={move} />
}

Use Theme

Styilsprovides a completely free theme experience. Apply them wherever you want as needed. 

import { createSystem } from '@styils/react'

const { styled, SystemProvider, useSystem } = createSystem({
  theme(mode) {
    return {
      color: mode === 'light' ? '#333' : '#fff',
      ...other // any
    }
  }
})

const Button = styled('div', (theme) => {
  return {
    color: theme.color
  }
})

render(() => {
  const { mode, setMode, theme } = useSystem()

  return (
    <SystemProvider>
      <Button onClick={() => setMode('dark')}>hello styils {mode}</Button>
    </SystemProvider>
  )
})

Server-side rendering

Styilssupportsssrand is easy to integrate into commonly used frameworks

If using themecreateExtractsshould be exported fromcreateSystem

View examples:NextandRemix 

import React from 'react'
import NextDocument, { Html, Head, Main, NextScript } from 'next/document'
import { createExtracts } from '@styils/react'
// or import { createExtracts } from 'yours-path'

export default class Document extends NextDocument {
  const { extractElement } = createExtracts()

  render() {
    return (
      <Html lang="en">
        <Head>
          {extractElement}
        </Head>
        <body>
          <Main />
          <NextScript />
        </body>
      </Html>
    );
  }
}

Animation Keyframes

Styilsprovides two ways to define as you like. 

const Foo = styled('div',{
  '@keyframes superAnimation': {
    '11.1%': {
      opacity: '0.9999'
    },
    '111%': {
      opacity: '1'
    }
  }
})
// or
import { keyframes } from '@styils/react'

const out = keyframes({
  from: {
    transform: 'rotate(0deg)'
  },
  to: {
    transform: 'rotate(360deg)'
  }
})

const Foo = styled('div',{
  animation: `13s ease 1.5s infinite none running ${out}`
})

Global Global

Styilsprovides two ways to define as you like. 

import { global, SystemProvider } from '@styils/react'

global({
  body: {
    backgroundColor: 'red'
  }
})
// or
global((theme) => ({
  body: {
    backgroundColor: theme.color
  }
}))
// or
const Foo = styled('div',{
  // Use in styled
  ':global':{
    body: { backgroundColor: 'red' }
  }
}

Query Media

Just like using media queries in regularCSS,@mediacan be placed directly in theCSSblock. 

const Root = styled('div', {
  maxWidth: 1280,
  margin: '0 auto',
  minHeight: '100vh',
  padding: '0 64px',
  transition: 'padding .3s',
  '@media screen and (max-width: 900px)': {
    padding: '0 28px'
  }
})
